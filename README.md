# Weather Website

A simple weather web application that displays real-time weather information using an external API.

## Purpose
Built to understand API integration and dynamic data handling in frontend applications.

## Tech Stack
- HTML  
- CSS  
- JavaScript  
- Weather API  

## Features
- Search weather by city  
- Displays temperature and weather conditions  
- Responsive layout  

## What I Learned
- API integration using JavaScript  
- Fetch API and asynchronous programming  
- DOM manipulation based on live data
